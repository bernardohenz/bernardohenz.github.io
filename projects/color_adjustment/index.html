Abstract

Image compositing aims to combine elements from multiple images producing natural-looking results. The Poisson image- editing framework can generate seamless compositions, but tends to introduce color changes in the inserted objects, which may generate unrealistic and unpleasing results. We present an efficient approach for controlling the amount of color preservation in the inserted objects, while ensuring a seamless transition between the foreground and background elements. Our technique modulates a Laplacian membrane according to some automatically computed parameter, which results in more naturally-looking compositions.
